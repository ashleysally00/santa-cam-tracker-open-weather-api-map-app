## Santa Cam Tracker

![Santa Cam Tracker](https://github.com/ashleysally00/santa-cam-tracker-open-weather-api-map-app/blob/main/santa-app-map.png)

The **Santa Cam Tracker** shows Santa's location on the map and calls the OpenWeather API to fetch and display the current weather for his location. Additionally, the sleigh is visible flying in the Santa Cam viewer for a magical experience.
